# First Group Game

末日废土幸存者肉鸽游戏设计项目。

## 游戏概念

玩家每局以普通幸存者身份开局，在开放废土小地图中探索、战斗、拾荒，并通过职业碎片拼装主动技能与被动能力。

游戏核心结构是：

- 开放探索与资源搜刮。
- 职业碎片拼装构筑。
- 9 个常规 Boss 的主动狩猎与等级追杀。
- 污染带来的高风险高收益成长。
- 100 级最终 Boss：异化的自己。

## 核心特色

- 普攻可自动或手动切换。
- 主动技能全部手动释放，最多携带 4 个。
- 被动碎片受污染负荷限制。
- 每个 Boss 都有专属精英怪和职业影响区。
- Boss 掉落物既是通关钥匙，也是构筑强化核心。
- 地图事件、资源点和 Boss 影响区互相关联。

## 首版原型范围

首版原型建议先实现：

- 1 张开放小地图。
- 3 个常规 Boss：厨师、小丑、快递员。
- 每个 Boss 2 到 3 种专属精英怪。
- 普攻自动/手动切换。
- 4 个主动技能槽。
- 被动污染负荷系统。
- 简化事件链。
- 30 级阶段目标。

## 本地运行

安装依赖：

```bash
npm install
```

启动原型：

```bash
npm run dev
```

验证：

```bash
npm test
npm run build
npm run e2e
```

原型操作：

- `WASD` 或方向键移动。
- `E` 搜刮附近资源点或事件点。
- `Q` 切换自动/手动普攻。
- `1` 到 `4` 释放主动技能。
- `X` 模拟击杀普通怪并获得经验。
- `B` 击杀当前追杀 Boss；没有追杀 Boss 时执行主动狩猎。
